# v0.6.3

> Released: 2026-04-08

## Fixes

- **Remote Control** — Fixed two critical bugs that prevented Remote Control from working correctly in production deployments.
- **Rate limiting** — Raised global API rate limit from 1000 to 3000 req/15 min; agent heartbeat and inbox polling routes are now exempt from the shared counter (same treatment as runner routes). Auth login limit (10/15 min) is unchanged.
