    ```
    default-src 'self'; script-src 'self' 'sha256-2oMWziIOZ3clEgoBlgQGv6iO60JrWMqbpz+42WnDniA='; style-src 'self' 'unsafe-inline'; img-src 'self' data: blob:; font-src 'self' data:; connect-src 'self' https://paperclip-paperclip-api.qiwa34.easypanel.host wss://paperclip-paperclip-api.qiwa34.easypanel.host; worker-src 'self'; frame-ancestors 'none'; base-uri 'self'; form-action 'self';
    ```

    IMPORTANT: The SHA-256 hash `sha256-2oMWziIOZ3clEgoBlgQGv6iO60JrWMqbpz+42WnDniA=` was computed from the exact text content between `<script>` and `</script>` tags in `ui/index.html` (the theme-detection inline script). The research document listed a different hash (`sha256-DBObfhsqKTQuFDHP5TJPKYUs76B8eG7yiBHiko5neM8=`) which is INCORRECT -- use the hash specified here.

    The complete vercel.json should be:
    ```json
    {
      "buildCommand": "pnpm --filter @paperclipai/ui build",
      "outputDirectory": "ui/dist",
      "framework": "vite",
      "routes": [
        { "handle": "filesystem" },
        { "src": "/(.*)", "dest": "/index.html" }
      ],
      "headers": [
        {
          "source": "/(.*)",
          "headers": [
            {
              "key": "Content-Security-Policy-Report-Only",
              "value": "default-src 'self'; script-src 'self' 'sha256-2oMWziIOZ3clEgoBlgQGv6iO60JrWMqbpz+42WnDniA='; style-src 'self' 'unsafe-inline'; img-src 'self' data: blob:; font-src 'self' data:; connect-src 'self' https://paperclip-paperclip-api.qiwa34.easypanel.host wss://paperclip-paperclip-api.qiwa34.easypanel.host; worker-src 'self'; frame-ancestors 'none'; base-uri 'self'; form-action 'self';"
            }
          ]
        }
      ]
    }
    ```

    Directive justification (add as JSON comment pattern -- since JSON does not support comments, no comment needed; the justification is documented here for the executor):
    - `script-src 'self' 'sha256-...'`: Hash covers the theme-detection inline script in index.html. All other scripts are ES modules from 'self'. 'unsafe-inline' MUST NOT appear. 'unsafe-eval' MUST NOT appear.
    - `style-src 'self' 'unsafe-inline'`: Required for Mermaid SVG inline style attributes. Tailwind v4 compiles to external CSS.
    - `img-src 'self' data: blob:`: data: for SVG mention chip CSS mask-image. blob: for createObjectURL download links.
    - `font-src 'self' data:`: Covers bundled fonts and data URIs.
    - `connect-src 'self' https://...api... wss://...api...`: Frontend API calls and WebSocket connections. Both https and wss MUST be listed -- they are different schemes.
    - `worker-src 'self'`: Service worker at /sw.js is same-origin. No blob: workers found in codebase.
    - `frame-ancestors 'none'`: Clickjacking protection.
    - `base-uri 'self'`: Prevents base tag injection.
    - `form-action 'self'`: No third-party form submissions.
  </action>
